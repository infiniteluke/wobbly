<h2 id="how-to-render">How To Render</h2>
<p>wobbly „Ä∞Ô∏è uses the child callback render function pattern. This is where you render whatever you want to based on the state of wobbly which is passed to the callback as parameters. The function is passed as the child prop of the Wobbly component:</p>
<pre><code class="lang-jsx">&lt;Wobbly&gt;
